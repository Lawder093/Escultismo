# Escultismo

Project Overview

This project was developed to make it easier and more accurate to share materials and keep track of activities between different Scout Development Centers.

The platform provides a centralized space where communities can publish content, manage information, and coordinate activities.

The goal is to create a scalable web application that improves communication and resource sharing across the scout network.


Features

Blog publishing system for scout activities and updates

Administrative dashboard for content management

Multi-view architecture supporting different scout centers

User session management and authentication

Structured content sharing between centers in Mexico, Colombia, and Argentina
Tech Stack

Tech Stack

Backend: Node.js, Express.js

Database: MongoDB, Mongoose

Frontend / Views: EJS, Express layouts

Authentication & Security

bcrypt, jsonwebtoken, express-session, connect-mongo
